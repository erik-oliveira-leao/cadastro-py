# Programa de Cadastro de Pessoas em Python

Este é um simples programa em Python que permite ao usuário cadastrar pessoas e realizar buscas em um menu interativo. 
## Como Funciona

### 1. **Cadastro de Pessoa**
O programa inicia com a opção de cadastrar uma nova pessoa. Ao selecionar essa opção, o usuário é solicitado a inserir o nome e a idade da pessoa, que são armazenados em uma lista como um dicionário.

### 2. **Busca de Pessoa**
O usuário pode buscar uma pessoa já cadastrada pelo nome. Se a pessoa existir, os detalhes (nome e idade) são exibidos. Caso contrário, uma mensagem de erro é mostrada.

### 3. **Menu Interativo**
O programa apresenta um menu com quatro opções:
- Adicionar Usuário
- Listar Usuários
- Buscar Usuário
- Sair

O menu continua sendo exibido até que o usuário escolha sair do programa.

Este programa é uma maneira simples e eficaz de entender como gerenciar registros básicos em Python. Você pode expandi-lo facilmente, adicionando novas funcionalidades, como a edição ou remoção de registros!
